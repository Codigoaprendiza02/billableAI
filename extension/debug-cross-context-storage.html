üéØ Context Information

Gmail Context (Content Script):

  • Runs in Gmail's origin (mail.google.com)
  • Has access to Gmail DOM elements
  • Stores data in chrome.storage.local (shared)
  • Also stores in localStorage (Gmail context only)

Extension Context (Popup):

  • Runs in extension's origin (chrome-extension://)
  • Has access to extension APIs
  • Can read from chrome.storage.local (shared)
  • Has separate localStorage (extension context only)
